This is based on the RI Tech Report Template that Daniel Morris wrote.

Miscellaneous Info: 
-I have a makefile for 
    compiling (make) 
    compiling + updating the bibliography (make full) 
    removing temp files (make clean)

-The watermark is put in if you include the watermark package, irrespective of
what commands you also include. Remove draftwatermark if you don't want a
watermark.

-You can add notes with \note{...}; you can then clear them by using the
second definition of note in the preamble
